# Instant Style - MVP 需求文档

## 1. 项目概述

- **项目名称**：Instant Style
- **类型**：图片风格转换 Web 应用
- **核心价值**：用户上传图片，快速应用6种视觉风格并下载
- **目标用户**：需要快速为图片添加风格化效果的用户

---

## 2. 功能需求

### 2.1 核心功能

| 序号 | 功能 | 描述 |
|------|------|------|
| F1 | 图片上传 | 支持拖拽或点击上传，支持 PNG/JPG/WEBP |
| F2 | 风格选择 | 6种预设风格可选（纯前端实现） |
| F3 | 风格转换 | 上传图片实时应用所选风格 |
| F4 | 结果预览 | 展示转换后的图片 |
| F5 | 图片下载 | 一键下载转换后的图片 |

### 2.2 6种风格（纯前端）

> 具体风格待开发时确定（如：黑白、复古、赛博朋克等）

### 2.3 用户流程

```
上传图片 → 选择风格 → 生成结果 → 下载图片
```
