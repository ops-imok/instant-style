# Instant Style

> 免费在线风格转换工具

## 功能特点

- 🎨 **6种风格**：简笔画、像素风、波普艺术、油画、漫画、复古
- 🔒 **隐私安全**：图片在浏览器本地处理，不上传服务器
- 💰 **完全免费**：无需API，无需付费
- ⚡ **即时处理**：实时预览效果

## 风格说明

| 风格 | 效果 |
|------|------|
| 简笔画 ✏️ | 照片变线条画 |
| 像素风 👾 | 照片变像素画 |
| 波普艺术 🎨 | 高对比色块 |
| 油画 🖼️ | 油画质感 |
| 漫画 💬 | 卡通效果 |
| 复古 📷 | 复古滤镜 |

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Canvas API

## 开发

```bash
npm install
npm run dev
```
